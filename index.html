Codecademy HTML semantic Cheat Sheet

HTML Semantics

Introduces meaning to a page through specific elements that provide context as to what is in between the tags.

A modern standard and makes a website accessible for people who use screen readers to translate the webpage and improves your website's SEO.

Tag Name Description
<main> Main Used to encapsulate the dominant content within a webpage.
<Footer> Footer The content at the bottom of the subject information.
<Article> Article Used to hold content that makes sense on its own.
<Section> Section Used to define elements in a document, such as chapters, headings, or any other document with the same name.
<Aside> aside Used to mark additional information that can enhance another elements but isn't required to understand the main content
<figure> figure An element used to encapsulate media such as an image, illustration, diagram, code snippet, etc.
<figcaption> figcaption Used to describe the media in the <figure> tag.
